# Saint Seiya Memory Game

### This is a simple memory game, made it as a Saint Seiya theme as it is my favorite anime.

## Objectives 

- Click a card and then click another one to match the first one.
- When both cards match you'll get a point and they will stay flipped.
- Try to match all cards with as little moves as poissible.
- CLick play at the bottom of the page to listen to the anima theme song while playing
